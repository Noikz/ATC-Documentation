ATC API documentation

ATC is the Swedish safety system used on trains, Norway uses the same system and there are also many variations of this system in diffrent countries. This API made in C was made to make it easier for developers to implement ATC into train simulators

Supported languages:

C/C++
JavaScript
Python
Lua

English Documentation:

int thisReturnsOne() // return 1 as a integer

Swedish Documentation:

int thisReturnsOne() // återvänder 1 som heltal/int
